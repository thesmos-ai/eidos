# Security policy

## Supported versions

eidos is under active development. Security fixes are issued against the
latest tagged release.

## Reporting a vulnerability

Please report security issues privately to **ai@stealthscale.io**.

Include:

- A description of the issue and its impact.
- Reproduction steps or a proof-of-concept.
- Affected version(s) or commit SHA(s).
- Any mitigations you've identified.

You will receive an acknowledgement within **3 business days**. We aim to issue
a fix or coordinated disclosure timeline within **30 days** of acknowledgement,
depending on severity and complexity.

Please do not file a public issue for security-relevant findings — open
disclosure puts users at risk before a patch is available. Once a fix is
released, we will credit the reporter (with their consent) in the release
notes.
